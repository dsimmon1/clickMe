Apocalypse, https://github.com/dsimmon1/clickMe

Apocalypse opens to a set of instructions and 3 buttons. If you click outbreak, 40 zombie buttons will appear on your sceen. If you click one zombie, it will cause an alert with the index. Clicking machete will cause one zombie button to disappear at a time. Clicking grenade will cause all the zombie buttons to disappear all at once. 

Prerequisites: 
	Internet connectivity - To load the application

Installing: 
    Clicking the link to application  

Deployment
	Application is deployed on Gh-Pages
